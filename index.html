Projects

What I've built

NutriLens

Problem

Manually logging every food item is tedious enough that most people abandon nutrition trackers within a few days — the friction is in the data entry, not the tracking itself.

Build

A Progressive Web App that cuts logging down to a barcode scan or a photo of a nutrition label. Barcode scans pull from Open Food Facts and USDA FoodData Central; for unscannable products, a vision LLM (Groq's Llama 4 Scout) reads a photographed label and converts the values into structured per-100g data automatically. Tracks macros, micronutrients, and daily goals, and installs like a native app via service worker caching.

Outcome

Turned food logging into a single scan or photo instead of manual search-and-entry.

Live demo GitHub

JavaScript · HTML/CSS · USDA FoodData Central API · Open Food Facts API · Groq Vision API · Service Worker/PWA · GitHub Pages
